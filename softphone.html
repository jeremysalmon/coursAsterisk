Configuration d'un softphone

Vous avez déclaré un périphérique SIP dans Asterisk. Vous avez donc créé un compte SIP qui peut être utilisé par un téléphone SIP ou par un softphone.

Nous allons donc maintenant télécharger et configurer un softphone avec le compte SIP que nous venons de déclarer dans Asterisk.

Il existe de nombreux softphone SIP sur le marché (voir : Liste des logiciels SIP sur Wikipedia). Nous allons utiliser Zoiper dans notre cas.

Commencez par télécharger Zoiper depuis le site Internet : http://www.zoiper.com/en/voip-softphone/download/zoiper3

Installez Zoiper. Au premier démarrage, veillez bien à autoriser Zoiper dans le Firewall (Windows vous le demandera).

La configuration d'un softphone ou d'un téléphone SIP requiert toujours au minimum les informations suivantes :

  • Nom d'utilisateur : le nom entre [] que vous avez déclaré dans sip.conf
  • Mot de passe : le mot de passe "secret" de sip.conf
  • Adresse IP (ou FQN) du serveur SIP : l'adresse de votre machine Linux (ifconfig eth0 dans la console de Linux si vous ne connaissez pas l'adresse du serveur).

Une fois que vous avez configuré votre compte dans le softphone la commande sip show peers dans la console d'Asterisk devrait vous indiquer 1 online

Félitations, vous avez un premier téléphone connecté à votre serveur de téléphonie.
