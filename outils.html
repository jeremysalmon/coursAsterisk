Votre système Debian est maintenant installé et démarré.

Durant tout nos TP, nous utiliserons cette machine.

Pour vous faciliter le travail et fonctionner que de vrais administrateurs Linux vous aurez à utilisez 3 logiciels

Putty

Putty est un client SSH qui và vous permettre de vous connecter depuis un logiciel Windows à la console de votre Linux.

Les avantages d'utiliser Putty par rapport à la console de VirtualBox :

  • copier / coller
  • défiler pour voir l'historique de la console
  • plusieurs consoles vers la même machine

Vous pouvez télécharger Putty ici : http://the.earth.li/~sgtatham/putty/latest/x86/putty.exe

L'idéal est de mettre putty.exe dans c:\windows\system32 comme celà il vous suffit de tapper "putty" dans le menu démarrer pour le lancer.

WinSCP

WinSCP est un client SSH qui vous permet d'accéder aux fichiers de votre serveur Linux directement depuis Windows.

Vous pouvez télécharger WinSCP ici :

Les avantages de WinSCP : https://winscp.net/eng/download.php

  • utilise le protocole SSH. Aucun besoin d'installer un serveur (ex : ftp) sur la machine Linux
  • permet d'éditer directement les fichier depuis Windows

Sublime Text

Sublime Text est un éditeur de texte léger et paramétrable

Vous pouvez télécharger Sublime Text ici : http://www.sublimetext.com/

Les avantages de Sublime Text :

  • léger
  • système de plugins complet
  • multi-curseur : (avec CTRL sans relacher et en cliquant que les ligns ou avec CTRL+D sur un mot)
