Statistiques téléphoniques dans MySQL

Pour faciliter la recherche des statistiques téléphonique, Asterisk vous permet de stocker les statistiques téléphoniques dans différents types de base de données. Le language SQL permettra de facilement retrouver des enregistrements.

A faire :

  • Installer MySQL sur votre serveur Debian
  • Créer la base de données pour vos statistiques téléphoniques (vous pouvez avoir de l'aide ici : voip-info.org)
  • Installer le paquet Debian qui permettra à Asterisk de gérer Mysql
  • Configurer le module cdr_mysql a l'aide de voip-info.org
  • Tester des appels et vérifier que vous avez des enregistrements dans MySQL
