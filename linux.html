Cours Asterisk

Raccourcis de la console Linux

  • CTRL+E = fin de ligne
  • CTRL+A = début de ligne
  • CTRL+U = supprime de fin au début
  • CTRL+K = supprime de début à fin
  • CTRL+R = recherche dans l'historique

Quelques commandes

  • pwd = répertoire en cours
  • whoami = avec quel utilisateur êtes vous connecté
  • who = liste des personnes connectées sur le système
  • history = historique des commandes
  • grep = cherche un chaine dans un fichier
    • ex : grep -n -r -i bonjour /etc/asterisk/*
    • -n : affiche le numéro de la ligne
    • -r : fait une recherche récursive dans les sous répertoires
    • -i : ne prend pas la casse en compte
  • tail = affiche les dernières lignes d'un fichier
    • ex : tail -n 100 /var/log/syslog = affiche les 100 dernières lignes du fichier syslog
    • ex : tail -f /var/log/systlog = affiche en temps réel la fin du fichier syslog
  • cat = affiche le contenu d'un fichier
    • ex : cat /var/log/syslog = affiche le contenu du fichier syslog
  • wc = compte le nombre de mots ou de lignes
    • ex : wc -l /var/log/syslog = affiche le nombre de lignes du fichier syslog
