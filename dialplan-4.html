4. Menu vocal interactif

Vous savez maintenant créer un diaplan, des extensions SIP, utiliser l'application Noop, Dial et Playback

Vous allez maintenant créer un menu vocal. Un menu vocal ou IVR est un accueil automatique ou le système vous lit un fichier vocal puis vous propose de faire un choix (1,2,3 ...).

Les menus vocaux sont utilisés par les entreprises qui désirent décharger la standardiste en permettant à l'appelant de directement choisir le service avec le quel il désire être mis en relation.

L'entrée dans un menu vocal déclenche la lecture d'un message vocal. L'application record vous permet d'enregistrer un message.

Créer un numéro 2000 qui permet d'enregistrer un message qui sera lu à l'entrée de votre menu vocal.

Vous avez besoin d'un micro pour enregistrer le message depuis votre softphone.

Avec Asterisk, l'application qui nous permettra d'attendre qu'un utilisateur appuie sur une touche est background

A l'aide de vos connaissances et d'Internet essayez de créer un menu vocal (numéro 902) qui

  • sur touche 1 : nous lit le fichier vocal demo-congrats
  • sur touche 2 : nous affiche un message "touche 2" dans la console de Asterisk
  • sur touche 3 : fait sonner le téléphone 101
