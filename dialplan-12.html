Statistiques téléphoniques

Les statistiques téléphoniques sont des éléments importants pour une entreprise. A partir de celles-ci, les entreprises peuvent connaître leurs consommations téléphoniques, savoir si elles ont des appels manqués, et avoir différents indicateurs sur l'activité téléphonique de leurs collaborateurs.

Asterisk gère nativement l'écriture de toutes ces informations dans un fichier texte. Vous pouvez trouver ce fichier dans /var/log/asterisk/cdr-csv/Master.csv

Vous trouverez le détails des différents champs sur la page la page voip-info.org dédiée

Vous pouvez également générer des fichiers personnalisés en éditant le fichier /etc/asterisk/cdr-custom.conf. Vous aurez alors un fichier généré dans /var/log/asterisk/cdr-custom/Mastercsv

Vous pouvez également généré des champs à partir du dialplan. Consultez core show function CDR dans la console d'Asterisk.

A FAIRE : Générer des champs personnalisé à partir du dialplan.
