De nouveaux équipements ... sur le réseau informatique existant

La mise en place d'un serveur de Téléphonie sur IP implique :

  • Serveur de ToIP : le serveur de téléphonie est un serveur informatique au même titre que le serveur de mail, le controleur de domaine ou un serveur de fichier.
    Le serveur de téléphonie peut donc être une machine physique ou une machine virtuelle au sein de l'infrastructure de virtualisation existante dans l'entreprise.
  • Téléphones IP : les téléphones IP sont des équipements qui se branchent sur le réseau informatique, et qui possèdent donc toutes les caractéristiques d'un périphériques IP.
    Ils existent de différentes marques de téléphones IP (snom, Avaya, Cisco, Grandstream, ...). Chaque téléphone à un mécanisme de configuration requérant au minimum un login, un mot de passe et l'adresse IP (ou le FQN) du serveur de téléphonie.
    Les téléphones IP embarque généralement un système d'auto-provisionning qui leurs permettent de se configurer automatiquement.
  • Softphone : est un logiciel qui transforme un ordinateur en téléphone IP. A l'instar des téléphones, il est nécessaire de posséder un login, un mot de passe et l'adresse IP d'un serveur de téphonie pour pouvoir profiter du système de ToIP directement depuis son ordinateur.
  • Lignes opérateur : les entreprises disposent toujours de lignes opérateur existantes.
    Ces lignes peuvent être de technologie analogique ou numérique.
    Ces lignes sont généralement a réutiliser dans le système de téléphonie sur IP, et doivent donc être connectées au serveur de téléphonie.
  • Passerelle : est un équipement qui permet de convertir le signal des lignes analogiques / numériques en flux IP compréhensibles par le serveur de téléphonie sur IP.
    Il existe de nombreuses marques de passerelles (Cisco, Avaya, Patton, Grandstream ...).
    Dans un environnement ou le serveur de téléphonie sur IP est virtualisé, on utilise ces passerelles afin de ne pas a avoir installer physiquement de matériel dans le serveur.
  • Carte PCI : le serveur de téléphonie sur IP Asterisk gère des cartes PCI de conversion des flux analogiques ou numériques des lignes opérateur en flux compatible.
    Il existe différentes marques de cartes PCI (Digium, Sangoma, Zycco).
    Ces cartes doivent être physiquement installées dans le serveur de téléphonie (ce qui peut les rendre difficile d'utilisation dans un environnement virtualisé).
    Les cartes PCI sont moins couteuses que les passerelles IP car au lieu d'intégrer des DSP de traitement du signal elle délègue la conversion au CPU du serveur.
    L'inconvénient des cartes PCI est qu'il faut généralement prévoir des serveurs plus puissants.
  • Opérateurs IP : les opérateur IP sont des opérateurs téléphoniques qui fournissent des liens téléphoniques au travers de connexion IP (généralement la connexion Internet de l'entreprise).
    Les opérateurs IP permettent plus de souplesses car il ne sont pas liés physiquement (comme le sont les lignes analogiques ou numériques) au serveur de téléphonie.
    Un opérateur IP fournir un login, un mot de passe et un FQN de serveur qui peut être utilisé dans le monde entier.
