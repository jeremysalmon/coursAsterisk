Cours Asterisk (Sommaire)

Raccourcis de la console Linux

  • CTRL+E = fin de ligne
  • CTRL+A = début de ligne
  • CTRL+U = supprime de fin au début
  • CTRL+K = supprime de début à fin
  • CTRL+R = recherche dans l'historique

Installation de Asterisk

L'installation de logiciel à l'aide du gestionnaire de paquets Debian est très simple.

Par exemple pour installer Asterisk et des exemples de fichiers de configuration, il vous suffit de taper dans votre console Linux :

apt-get install asterisk asterisk-config asterisk-core-sounds-fr

Pour pouvoir effectuer des installations sur votre machine Linux il faut être "root". Si vous êtes connecté avec votre utilisateur, il suffit de taper su et de saisir le mot de passe administrateur pour passer en mode root.

Félitations, vous avez un serveur Asterisk installé sur votre ordinateur.
