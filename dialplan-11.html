Système de centre d'appel

Asterisk permet de mettre en place des centres d'appel efficaces grâce à une gestion des agents et des files d'attente souple.

Généralement un centre d'appel se compose :

  • D'agents : qui prennent ou emettent des appels
  • De files d'attente : qui distribuent les appels aux différents agents
  • De statistiques : qui permettent d'avoir une visibilité sur appels effectués

Gestion des agents

Un agent est un compte qui va pouvoir se connecter, déconnecter et se mettre en pause.

Toutes ces opérations seront effectuées à partir du dialplan

Pour créer un agent il faut ouvrir le fichier /etc/asterisk/agents.conf

Pour créer un agent, dans le contexte [agents] ajoutez :

agent => 1000,1234,Jeremy Salmon
  • 1000 : le numéro de l'agent
  • 1234 : le mot de passe de l'agent
  • Jeremy Salmon : son nom

Dans la console d'Asterisk vous pouvez utilisez agent show pour voir vos agents et leur status.

A FAIRE : Utiliser les fonctions agentLogin pour connecter un agent

Gestion des files d'attente

Une file d'attente (queue) est une "salle" virtuelle dans laquelle sont connectés des agents et qui attendent des appels.

Quand un appel arrive, il est distribué suivant une "stratégie" à un ou plusieurs agents.

Au sein d'une file d'attente un agent peut être connecté ou en pause.

Pour créer une file d'attente il faut éditer le fichier /etc/asterisk/queues.conf

[maFile]
musiclass = default
strategy = ringall
timeout = 10
retry = 5
joinempty = yes
leavewhenempty=yes
context = interne
member => Agent/1000
member => Agent/1001

La console d'Asterisk vous permet de manipuler les files. Par exemple queue show vous affichera le détail de toutes les files configurées sur le système

Plan d'appel

Le plan d'appel doit maintenant nous permettre de connecter de faire entrer un appel dans une file d'attente.
